# 🎬 IMDB Movie App (MSCS Project)

A modern movie browsing web application built using **React + Redux Toolkit** that allows users to explore trending movies, search films, and manage movie data using TMDB API.

This project was developed as part of my **Master of Science in Computer Science (MSCS)** learning journey.

## 📌 Features

- 🔥 Trending Movies
- 🔎 Movie Search
- 📄 Pagination
- ⭐ Movie Details
- 🧠 Global State Management using Redux Toolkit
- ⚡ Fast UI with Vite
- 🎨 Responsive UI using Tailwind CSS
- 🌐 API Integration (TMDB API)

---

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- Vite
- Tailwind CSS
- Axios
- TMDB API
- JavaScript (ES6+)
